Converts SOCKS server requests to Wisp requests

```
Wisp to SOCKS5

Usage: wisp2socks --wisp <URL> --socks <ADDRESS>

Options:
  -w, --wisp <URL>       WebSocket URL of Wisp server
  -s, --socks <ADDRESS>  Address to listen for SOCKS server
  -h, --help             Print help
```
